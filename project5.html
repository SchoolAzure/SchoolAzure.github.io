Project Write Up
One difficulty we had was getting the while loop done correctly. Previously, the program ran through a sequence of events that looped over. Which left the user unable to choose whether they wanted to start with decoding or encrypting. This was solved by setting a loop to a variable, and letting the user select an option. Selecting exit would end the loop and stop the program. The portion I did myself was the setup of the while loop, setting it up as a selection.
